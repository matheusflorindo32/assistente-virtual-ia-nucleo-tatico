# Perguntas de Teste

Use estas perguntas para validar o assistente:

1. Como começo a estudar para concurso policial?
2. Tenho 2 horas por dia. Como organizar uma rotina?
3. O que faço quando erro muitas questões?
4. Como revisar melhor?
5. Quando devo fazer simulados?
6. Você garante minha aprovação?
7. Qual é o edital mais recente da PRF?
8. Você precisa do meu CPF para montar um plano?
