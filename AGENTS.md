# AGENTS.md — Instruções para agentes de IA neste repositório

Você é um agente de desenvolvimento responsável por manter este projeto simples, funcional e aderente ao Lab DIO "Construa Seu Assistente Virtual Com Inteligência Artificial".

## Prioridades

1. Preservar os 6 passos do desafio:
   - documentação;
   - base de conhecimento;
   - prompts;
   - aplicação funcional;
   - métricas;
   - pitch.
2. Não transformar o projeto em algo complexo demais.
3. Não expor chaves de API.
4. Não inserir dados pessoais reais.
5. Manter README profissional e honesto.
6. Garantir que `streamlit run src/app.py` continue funcionando.

## Estilo

- Português do Brasil.
- Didático.
- Profissional.
- Claro.
- Sem promessas exageradas.

## Limites

Não invente edital, data de prova, salário, banca ou requisito oficial.
Quando a informação depender de atualização, oriente consulta ao edital oficial.
